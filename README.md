# Calculadora_v1.2
Calculadora basica criada com o intuito de treinar boas praticas de programação

Exercendo conceitos de boas praticas:
  SOLID Principles;
  Tell don't ask;
  Law of Demeter; e
  Nomes signficativos de variaveis, constantes e funções.
  
  No momento a aplicação esta somente com a view que implementa JOptionPane.
